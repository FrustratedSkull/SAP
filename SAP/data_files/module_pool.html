Module Pool Programming

By Nikhil| Date: 15 feb, 2019 | 0 Comments

Level: Intermediate


Steps to work with module pool program: -

  • Create a module pool program (in SE38) & implement the logic.
  • Design the required screens (in SE51) & attached to the program.
  • Design the required menus (in SE41) & attached to screen.
  • Design the database tables (in SE11) & as per client requirement.
  • Create the transaction code (in SE93) & run the module pool program.

    These 5 steps, we can work with single transaction. I.e. SE80 (Object Navigator)
